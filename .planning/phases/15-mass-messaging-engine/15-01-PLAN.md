**Selection Algorithm:**

```typescript
interface ChipScore {
  chipId: string;
  score: number;
  reasons: string[];
}

/**
 * Select the best chip for sending a message.
 * 
 * Scoring factors:
 * 1. Health status (healthy=100, degraded=50, others=0)
 * 2. Available capacity (daily limit - sent today)
 * 3. Hourly capacity remaining
 * 4. Segment affinity (if chip has been used for this segment before)
 * 5. Least recently used (to distribute load)
 * 6. Error count penalty
 */
export async function selectBestChip(
  segmentId?: string
): Promise<{ chip: Chip | null; reason: string }>

/**
 * Get all chips that can send right now.
 * Conditions:
 * - healthStatus in ('healthy', 'degraded')
 * - messagesSentToday < dailyLimit
 * - messagesSentThisHour < hourlyLimit
 * - enabled = true
 */
export async function getAvailableChips(): Promise<Chip[]>

/**
 * Check if chip can send more messages.
 */
export function canChipSend(chip: Chip): boolean

/**
 * Get total available capacity across all healthy chips.
 */
export async function getTotalAvailableCapacity(): Promise<{ daily: number; hourly: number }>
```

**Selection Logic:**
1. Load all available chips (healthy/degraded, enabled, has capacity)
2. If segmentId provided, prefer chips with assignedSegments matching
3. Score each chip:
   - healthStatus: healthy=100, degraded=50
   - daily capacity % remaining * 30
   - hourly capacity % remaining * 20
   - segment affinity bonus: +50
   - error count penalty: -10 per error
   - least recently used bonus: +10 (based on lastHealthCheck)
4. Return highest scoring chip

If no chips available, return null with reason explaining why (all banned? all at capacity? all disconnected?)

**Configuration (environment or defaults):**

**Flow:**

```typescript
export async function GET(request: NextRequest) {
  // 1. Auth check (CRON_SECRET or loopback)
  
  // 2. Check time window (only send during configured hours)
  const now = new Date();
  const hour = now.getHours();
  if (hour < TIME_WINDOW_START || hour >= TIME_WINDOW_END) {
    return NextResponse.json({ 
      message: 'Outside time window, skipping',
      window: `${TIME_WINDOW_START}:00-${TIME_WINDOW_END}:00`
    });
  }
  
  // 3. Get next batch of queued messages
  const messages = await getNextQueuedMessages(BATCH_SIZE);
  if (messages.length === 0) {
    return NextResponse.json({ message: 'Queue empty' });
  }
  
  // 4. Load config (evolution API)
  const config = await loadConfig();
  
  // 5. For each message:
  for (const msg of messages) {
    // a. Select best chip
    const { chip, reason } = await selectBestChip(msg.segmentId);
    
    if (!chip) {
      await markMessageFailed(msg.id, `No chip available: ${reason}`);
      continue;
    }
    
    // b. Assign and mark as sending
    await assignMessageToChip(msg.id, chip.id);
    await markMessageSending(msg.id);
    
    // c. Random delay before sending (anti-ban)
    const delay = randomInt(MIN_DELAY_MS, MAX_DELAY_MS);
    await sleep(delay);
    
    // d. Send via Evolution API
    try {
      const result = await sendText(
        config.evolutionApiUrl,
        config.evolutionApiKey,
        chip.instanceName,
        msg.voterPhone,
        msg.resolvedMessage,
        { delay: randomInt(2000, 5000) } // typing simulation
      );
      
      // e. Mark as sent
      await markMessageSent(msg.id, result.key.id);
      
      // f. Increment chip counters
      await incrementChipCounter(chip.id, 'messagesSentToday');
      await incrementChipCounter(chip.id, 'messagesSentThisHour');
      
    } catch (err) {
      await markMessageFailed(msg.id, String(err));
    }
  }
  
  // 6. Return summary
  return NextResponse.json({
    processed: messages.length,
    window: `${TIME_WINDOW_START}:00-${TIME_WINDOW_END}:00`
  });
}
```
