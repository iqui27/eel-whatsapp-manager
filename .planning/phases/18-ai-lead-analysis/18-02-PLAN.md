# Phase 18 Plan 02: Batch Profiling Cron + AI Insights Panel

## Objective

Build the nightly batch profiling cron for lead scoring and the AI insights panel for the CRM voter profile page.

## Context

- AI analysis module from 18-01
- Voter engagement scores exist
- Need automated profiling and UI display

## Tasks

### Task 1: Create batch profiling cron
type: auto

**File:** `src/app/api/cron/ai-profile/route.ts`

**Logic:**
1. Find voters without recent AI analysis (> 7 days)
2. Load voter data + message history
3. Call Gemini profileLead() for each
4. Update voter: aiTier, aiSentiment, aiRecommendedAction
5. Log results

**Rate limiting:**
- Max 100 voters per run
- 1 second delay between API calls
- Store last run timestamp

**Verification:** Cron endpoint compiles.

---

### Task 2: Create AI insights helper
type: auto
