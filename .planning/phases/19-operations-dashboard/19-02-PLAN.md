# Phase 19 Plan 02: Group Capacity Grid + Conversion KPIs + Message Feed

## Objective

Build the group capacity grid, conversion KPIs section with sparklines, and real-time message feed for the operations dashboard.

## Context

- Group data from Phase 16
- Conversion data from Phase 17
- Message queue from Phase 15

## Tasks

### Task 1: Create group capacity grid component
type: auto

**File:** `src/components/group-capacity-grid.tsx`

**Display:**
- Grid of group cards
- Each card shows:
  - Group name with status badge
  - Capacity progress bar (current/1024)
  - Copy invite link button
  - Quick-create overflow button (if near capacity)
  - Chip assignment

**Verification:** Component renders.

---

### Task 2: Create conversion KPIs component
type: auto

**File:** `src/components/conversion-kpis.tsx`

**Display:**
- KPI cards in a row:
  - Total Sent
  - Delivery Rate %
  - Read Rate %
  - Reply Rate %
  - Group Join Rate %
- Each card has:
  - Large number
  - Sparkline (last 7 days)
  - Trend indicator (↑↓→)

**Verification:** Component renders.

---

### Task 3: Create message feed component
type: auto

**File:** `src/components/message-feed.tsx`

**Display:**
- Scrolling feed of recent messages (last 50)
- Each message shows:
  - Direction icon (inbound/outbound)
  - Chip name
  - Lead name/phone
  - Message preview (truncated)
  - Status icon (sent/delivered/read/failed)
  - Timestamp
- Auto-refresh every 10s
- Pause on hover

**Verification:** Component renders.

---
