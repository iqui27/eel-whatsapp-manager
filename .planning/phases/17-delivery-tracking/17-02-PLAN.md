# Phase 17 Plan 02: Conversion Funnel UI + Campaign Analytics

## Objective

Build the conversion funnel visualization component and campaign detail page with delivery timeline, per-chip breakdown, and analytics.

## Context

- Conversion events tracked in Phase 17-01
- Campaign stats already include sent/delivered/read/replied counters
- Need visual representation of the funnel

## Tasks

### Task 1: Create conversion funnel API
type: auto

**File:** `src/app/api/campaigns/[id]/funnel/route.ts`

**GET endpoint:**
```typescript
{
  campaignId: string;
  total: number;
  sent: number;
  delivered: number;
  read: number;
  replied: number;
  clicked: number;
  joinedGroup: number;
  converted: number;
  percentages: {
    sent: number;
    delivered: number;
    read: number;
    replied: number;
    joinedGroup: number;
  };
}
```

**Verification:** API returns funnel data.

### Task 5: Create campaign analytics API
type: auto

**File:** `src/app/api/campaigns/[id]/analytics/route.ts`

**GET endpoint:**
```typescript
{
  campaign: Campaign;
  funnel: FunnelData;
  timeline: DeliveryEvent[];
  chipBreakdown: ChipBreakdown[];
  alerts: Alert[];
}

interface Alert {
  type: 'warning' | 'error';
  message: string;
  chipId?: string;
}
```

**Alerts:**
- High block rate (>5%)
- Chip with high failure rate (>10%)
- Low delivery rate (<80%)
- Campaign stalled (no sends in 1 hour)

**Verification:** API returns analytics data.
