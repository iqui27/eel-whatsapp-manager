# Plan 21-04: Chip Failover Enhancement

## Objective
Melhorar o sistema de failover para trocar de chip automaticamente quando queimar, sem perder mensagens.

## Tasks

### Task 1: Enhance `selectBestChip()` with fallback chain
type: auto

**Implementation:**
1. Add `fallbackChipId` to segments or chips
2. When selecting chip, check for fallback
3. Prefer same-segment chip as fallback
4. Log fallback activation

**Files:**
- `src/lib/chip-router.ts`

type: auto

2. Reassign pending messages to new chip
3. Update active campaigns
4. Send notifications
5. Log failure event

**Files:**
- `src/lib/chip-failover.ts` (new file)
