# Plan 21-05: Setup Wizard Implementation

## Objective
Criar wizard guiado para configuração inicial do sistema, permitindo que novos usuários configurem tudo em menos de 10 minutos.

## Tasks

### Task 1: Create wizard shell with step navigation
type: auto

**Implementation:**
1. Create `/setup` page with step indicator
2. Create progress persistence (localStorage + DB)
3. Add step validation before proceeding
4. Add back/skip functionality
5. Track completion percentage

**Files:**
- `src/app/setup/page.tsx`
- `src/app/setup/layout.tsx`
- `src/components/setup-wizard-nav.tsx`

---

**Files:**
- `src/app/setup/chips/page.tsx`
- `src/components/chip-qr-scanner.tsx`
